# 게임 컨셉 기획서: Sweet Crunch

## 1. 게임 개요

| 항목 | 내용 |
|---|---|
| **타이틀 (가칭)** | Sweet Crunch |
| **장르** | Match-3 캐주얼 퍼즐 |
| **테마** | 디저트 / 베이킹 |
| **아트 스타일** | 미니멀 플랫 일러스트 |
| **플랫폼** | Android / iOS |
| **타겟 유저** | 25~45세, 캐주얼 게이머 (여성 비율 높음) |
| **세션 길이** | 1회 2~3분 (레벨 당) |
| **수익 모델** | 광고 (보상형) + IAP (부스터, 라이프) |

---

## 2. 컨셉 키워드

> **"한 입 크기의 달콤한 퍼즐"**

- **Cozy** — 따뜻한 색감, 스트레스 없는 플레이
- **Minimal** — 군더더기 없는 깔끔한 비주얼
- **Satisfying** — 매칭/폭발 시 주스감(juice) 있는 피드백
- **Bite-sized** — 짧은 세션, 가볍게 즐기는 플레이

---

## 3. 아트 디렉션

### 3.1 아트 스타일

- **플랫 디자인** — 그림자 최소화, 단순한 형태, 굵은 아웃라인 없음
- **둥근 형태** — 모든 디저트 타일은 둥글둥글한 실루엣
- **소프트 그라데이션** — 완전 플랫보다는 미세한 그라데이션으로 입체감 부여
- **배경** — 크림색/베이지 톤의 단색 또는 미세한 패턴 (제과점 타일, 린넨 텍스처)

### 3.2 컬러 팔레트

| 용도 | 컬러 | Hex (참고) |
|---|---|---|
| **배경** | 웜 크림 | `#FFF8F0` |
| **보드 배경** | 소프트 베이지 | `#F5E6D3` |
| **타일 1 — 컵케이크** | 소프트 핑크 | `#FF9BAA` |
| **타일 2 — 도넛** | 라벤더 | `#C4A7E7` |
| **타일 3 — 마카롱** | 민트 그린 | `#A8E6CF` |
| **타일 4 — 크루아상** | 웜 골드 | `#FFD07B` |
| **타일 5 — 아이스크림** | 스카이 블루 | `#89CFF0` |
| **타일 6 — 초콜릿** | 코코아 브라운 | `#A0725B` |
| **강조색 (점수, 콤보)** | 코랄 | `#FF6F61` |
| **텍스트** | 다크 브라운 | `#4A3728` |

> 파스텔 톤 위주이되 타일 간 **명도/채도 차이를 충분히** 두어 색약 유저도 구분 가능하도록 설계

### 3.3 타일 디자인 (6종)

각 타일은 **64x64px** 기준 제작, 레티나 대응 **128x128px** 원본.

| 타일 | 디저트 | 형태 특징 | 주 색상 |
|---|---|---|---|
| 1 | **컵케이크** | 둥근 크림 위 체리 | 핑크 |
| 2 | **도넛** | 원형 + 가운데 구멍 | 라벤더 |
| 3 | **마카롱** | 납작한 원 2개 겹침 | 민트 |
| 4 | **크루아상** | 초승달 형태 | 골드 |
| 5 | **아이스크림** | 콘 위 둥근 스쿱 | 블루 |
| 6 | **초콜릿** | 사각 + 격자 무늬 | 브라운 |

> 형태(실루엣)만으로도 구분 가능해야 함 — 색상 + 형태 이중 구분

## 4. 게임 메커니즘

### 4.1 기본 규칙

| 항목 | 값 |
|---|---|
| 그리드 크기 | **8x8** (기본) |
| 셀 크기 | 64x64 px (논리), 화면에 맞게 스케일링 |
| 매칭 조건 | 같은 종류 3개 이상 수평 또는 수직 연속 |
| 스왑 방식 | 인접 타일과 스왑 (터치 드래그 또는 탭-탭) |
| 유효하지 않은 스왑 | 매칭이 발생하지 않으면 원위치 복귀 |
| 이동 제한 | 레벨별 제한 횟수 (무제한 시간 아님) |

### 4.2 특수 타일

| 매칭 조건 | 생성되는 특수 타일 | 이름 | 효과 |
|---|---|---|---|
| **4개 일렬** | 줄무늬 타일 | Rolling Pin (밀대) | 가로 또는 세로 한 줄 전체 제거 |
| **L자 또는 T자** | 폭탄 타일 | Oven Bomb (오븐 폭탄) | 주변 3x3 범위 제거 |
| **5개 일렬** | 레인보우 타일 | Golden Whisk (황금 거품기) | 보드 위 선택한 종류의 타일 전체 제거 |

### 4.3 부스터 (유료/보상형)

| 부스터 | 효과 | 획득 방법 |
|---|---|---|
| Extra Moves +5 | 이동 횟수 5회 추가 | 레벨 클리어 보상 / IAP |
| Spatula (뒤집개) | 타일 1개 자유 제거 | 광고 시청 / IAP |
| Mixer (믹서) | 보드 전체 셔플 | 광고 시청 / IAP |

## 5. 메타게임: 디저트 도감

> 무거운 꾸미기/스토리 대신 **가벼운 수집 시스템**으로 리텐션 확보

### 5.1 구조

- 레벨을 클리어하면 **레시피 조각**을 획득
- 조각을 모으면 **디저트 레시피가 완성**됨
- 완성된 레시피는 **도감(Recipe Book)**에 등록
- 도감에는 디저트 일러스트 + 간단한 설명 표시

### 5.2 도감 카테고리 (예시)

| 카테고리 | 디저트 수 | 해금 구간 |
|---|---|---|
| 프랑스 클래식 | 8종 | 레벨 1~40 |
| 일본 와가시 | 8종 | 레벨 41~80 |
| 한국 전통 디저트 | 8종 | 레벨 81~120 |
| 이탈리안 돌체 | 8종 | 레벨 121~160 |
| 아메리칸 스위트 | 8종 | 레벨 161~200 |

> 각 카테고리 완료 시 **배경 테마 변경** (보드 배경색, 분위기)

## 6. 레벨 시스템

### 6.1 목표 유형

| 목표 타입 | 설명 | 예시 |
|---|---|---|
| **점수 달성** | 목표 점수 이상 획득 | "15,000점 달성" |
| **특정 타일 수집** | 지정 타일 N개 매칭 | "컵케이크 30개 제거" |
| **장애물 제거** | 장애물 타일 제거 | "얼음 타일 15개 깨기" |
| **레시피 재료 수집** | 특수 재료 타일 보드 아래로 떨어뜨리기 | "밀가루 3개, 설탕 2개 수집" |

### 6.2 장애물 (점진적 추가)

| 장애물 | 등장 레벨 | 설명 |
|---|---|---|
| **Ice (아이싱)** | 레벨 11~ | 타일 위 얼음 코팅. 1회 매칭으로 제거 |
| **Double Ice** | 레벨 31~ | 2회 매칭 필요 |
| **Chocolate Spread** | 레벨 51~ | 매 턴 인접 타일로 퍼짐. 매칭으로 제거 |
| **Cookie Jar (유리병)** | 레벨 71~ | 내부 타일 잠금. 인접 매칭 2회로 해제 |
| **Waffle Grid** | 레벨 91~ | 해당 칸에서 매칭해야 와플 제거 (2단계) |

## 7. 화면 구성 (Stitch 디자인 대상)

### 7.1 필요 화면 목록

|---|---|---|
| 1 | **타이틀/메인 메뉴** | 로고, 플레이 버튼, 설정, 도감 버튼 |
| 2 | **레벨 선택 (맵)** | 레벨 번호 노드, 현재 진행도, 카테고리 배경 |
| 3 | **레벨 시작 팝업** | 목표 표시, 부스터 선택, 시작 버튼 |
