## Band 1 (L1~50) 기본 공식

```typescript
// levels.ts:buildLevel() — getBand(i).id === 'world_01' 분기
const t = (i - 1) / 49;
moves = Math.round(30 - 12 * Math.pow(t, 0.6));   // 30 → 18
goalCount = Math.round(12 + 35 * Math.pow(t, 0.8)); // 12 → 47
baseStar = 2000 + Math.round(7000 * t);               // 2000 → 9000
```

마일스톤 보너스: `i % 10 === 0` → moves +5, `i % 5 === 0` → moves +3
L36-40 추가 보너스: moves +3 (멀티 목표 구간 완충)

> 구조 변경: `levels.ts`는 `getBand(i)`, `getPod(i)` 호출로 Band/Pod 스펙을 실제 생성 흐름에 연결.
> 보드 레이아웃은 Pod별 전용 함수(`applyPod01Board`~`applyPod10Board`)로 분리됨.

## Pod 구간별 특성 및 조정 내역

- **타일 수**: 4 (L1-5 하드코딩 4색)
- **특이사항**: L1-5 개별 하드코딩 (`levels.ts:88~109`)

- **밸런스 상태**: ✅ 안정 — 손 대지 말 것

- **타일 수**: 5
- **보드**: L6(중앙 2x2 얼음), L7(코너+중앙), L8-10(상하 가장자리 행)
- **튜토리얼**: L6(얼음 2스텝)
- **밸런스 상태**: ✅ 안정

### Pod 03 — 체인 (L11~15)

### Pod 07 — 체인+돌 혼합 (L31~35)
- **타일 수**: 6
- **보드**: `mixedBoard([], chain중앙, stone코너, 1겹, 2겹)` — 중앙 체인 + 코너 돌 2겹
- **P3 조정**: L31-34 moves +1
- **밸런스 상태**: ⚠️ 요주의 — 돌 2겹 + 체인 조합이 이동 수 대비 빡빡할 수 있음, 플레이테스트 요망

### Pod 08 — 멀티 목표 입문 (L36~40)
- **타일 수**: 6
- **보드**: `iceBoard(중앙8셀, 3겹)` — 얼음 3겹 패치 (3겹은 특수젬 필수)
- **목표**: collect + clearIce (이중 목표)
- **P3 조정**: L36-39 goalCount -3, L38-39 moves +1, L36-40 공식에 moves +3 보너스
- **튜토리얼**: L36(멀티 목표 2스텝 — "이번엔 목표가 두 개예요")
- **밸런스 상태**: ⚠️ 요주의 — 얼음 3겹은 특수젬 없이 실질적으로 불가능, 특수젬 생성 유도 설계 검토 필요

### Pod 09 — 전체 압박 + 타이머 (L41~45)
- **타일 수**: 6
- **보드**: `mixedBoard(얼음8셀x2겹, chain4셀, stone4셀, 2겹, 1겹)` — 모든 장애물 혼합
- **목표**: collect * 0.7 + clearIce 8개
- **P3 조정 (중요)**:
  - L41-44 moves = **21** 하드셋 (공식 15~18 → 21로 강제 상향)
  - L41-44 goalCount -2
  - **타이머 젬**: L43 → L44로 이동 (기존 L43 도입, P3에서 L44로 연기)
  - L44-45: 타이머 2개 turns=6
- **튜토리얼**: L43("보드는 모든 장애물이 한꺼번에"), L44(타이머 젬 첫 도입)
- **밸런스 상태**: ⚠️ 요주의 — L43은 moves=21로 플레이어에게 준비 기간 제공, L44 타이머 처음 접하므로 실제 폭발 시 반응 확인 필요

### Pod 10 — 하드코어 (L46~49) + 마일스톤 (L50)
- **타일 수**: 6
- **보드**: `mixedBoard(얼음8셀x3겹, chain4셀, stone6셀x2겹, 3겹, 2겹)` — 최고 난이도
- **목표**: collect * 0.6 + clearStone 6개

  - L47-49 moves +2
  - 타이머: L46 없음, L47-48 2개 turns=6, L49 4개 turns=6
- **튜토리얼**: L46(stone 목표 + 특수젬 조합 2스텝)
- **밸런스 상태**: 🔴 검증 필요 — clearStone 6개 + 3겹 얼음 + 타이머 조합은 이 게임 최고 난이도

## 알려진 밸런스 리스크

### 🔴 고위험
1. **L36-40 얼음 3겹**: 특수젬 없이 클리어 불가능에 가까움. 수집 목표와 동시에 달성해야 하므로 이동 수 압박 극심. → 3겹 셀 수 줄이거나 moves +2 추가 검토
2. **L46-49 clearStone 6개**: 돌 2겹 6개 = 12번 인접 매칭 필요 + 3겹 얼음 동시 공략. P3에서 moves +1~2 했지만 여전히 빡빡할 가능성

### ⚠️ 중위험
3. **L31-35 체인+돌2겹 혼합**: 기본 공식 moves가 낮은 구간(L31 약 23이동). P3 +1 적용됨, 플레이테스트 요망
4. **L44 타이머 젬 첫 도입**: 조작 방법을 모르는 플레이어는 2-3번 실패 예상. L44 전에 튜토리얼이 뜨지만 실제 체감 난이도 확인 필요
5. **L100 오버라이드**: 3목표 동시 달성, Band 2 최종 마일스톤. 28이동으로 여유롭게 설계되었으나 미검증

### ✅ 검증된 안정 구간
- L1-20: 개별 조정 + 단순 장애물 도입 → 안정적
- L50 마일스톤: 오버라이드로 타이머 제거 + moves 25 → 쉬운 마일스톤
